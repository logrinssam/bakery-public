# 추후 기능 (아이디어 메모)

## 명예의 전당 — 오늘 등록 전광판

**상태:** 보류 (현재는 기존 랭킹·명단 UI 유지)

**아이디어:** 오늘 명예의 전당에 등록한 사람이 **전광판(마키/티커)** 형태로 화면에 바뀌며 노출된다.

- 실시간에 가깝게 “오늘 등록자”를 강조
- 기존 TOP 정렬·50명 fetch·시간대 스냅샷 랭킹과는 별도 레이어로 두는 방향 검토
- 구현 시: `hallRecords.createdAt` / 등록일(KST) 기준 필터 + 순환 애니메이션 UI

**결정 (2026-05):** 당분간 현재 방식 지켜보기. 위 기능은 추후 적용.
